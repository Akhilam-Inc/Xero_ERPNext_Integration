Overview

The Xero Integration App seamlessly connects your Frappe/ERPNext system with Xero accounting software

Automated Sync

Automatically sync customers, suppliers, items, and transactions between systems

Real-time Updates

Keep both systems synchronized with real-time data updates

Error Reduction

Eliminate manual data entry errors with automated processes

Audit Trail

Maintain complete transaction history and mapping records

Key Benefits:
  • Automated Data Sync between Frappe/ERPNext and Xero
  • Real-time Updates for immediate data consistency
  • Error Reduction through automated processes
  • Time Saving by reducing administrative overhead
  • Complete Audit Trail for all transactions

Prerequisites

System Requirements

  • Frappe Framework v15.x or ERPNext v15.x
  • Internet connectivity for API communication
  • Administrator access to both Frappe/ERPNext and Xero

Xero Requirements

  • Xero organization with appropriate permissions
  • API access enabled
  • Valid Xero subscription (Starter, Standard, or Premium)
Important: Ensure you have administrator privileges in both systems before proceeding with the installation.

Installation & Setup

Install the App

Run the following commands in your Frappe bench directory:

bench get-app xero_erpnext_integration
bench install-app xero_erpnext_integration --site your-site-name

Enable the Integration

  1. Login to your Frappe/ERPNext system
  2. Go to Xero Integration → Xero Settings
  3. Click

Xero Authentication

  1. In Xero Settings, Set client ID, client Secret, Enable, and Debug Mode
  2. click
  3. You'll be redirected to Xero's authorization page
  4. Login with your Xero credentials
  5. Grant permissions to the integration app
  6. You'll be redirected back with a success message
Success! Once authentication is complete, you'll see a green connection status in your Xero Settings.

Features

Customer Synchronization

From Frappe/ERPNext to Xero

  • Create Contact for customer to sync with Xero
  • Sales Invoice
  • Cancel Sales Invoice
Usage:
  1. Create contact in contact page, check secnd to Xero box, then click on
  2. To also create Contact ion sales invoice, click
  3. On Submit of Sales Invoice, If Do not Sync to Xero is checked, it will not be synced to Xero for payment
  4. Check the Xero API Log for api status
  5. Confirm Sales Invoice in Xero, if Do not Sync to Xero is uncheckked
  6. Cancel Sales Invoice in Frappe/Erpnext, if successful, then confirm in Xero.

From Xero to Frappe/ERPNext

  • Payment Entries
  • Void Sales Invoice
  • Create Contact to sync with Erpnext
Usage:
  1. Create or update a contact in Frappe/Erpnext
  2. Create sales invoice in Frappe/Erpnext
  3. Make payment for sales invoice in Xero
  4. Check the Xero API Log for api status
  5. Confirm sales invoice paid after payment in xero
  6. If sales invoice is voided in xero, also confirm its reflected.
  7. Check the Xero Payment Scheduler for sync status

Troubleshooting

Common Issues

1. Authentication Errors

Problem: "Invalid token" or "Authentication failed"
Solution:
  1. Go to Xero Settings
  2. Click to reconnect
  3. Complete the authentication process
  4. Test the connection

2. Sync Failures

Problem: Records not syncing between systems
Solution:
  1. Check Xero API Log for errors
  2. Verify scheduler is running
  3. Ensure required fields are populated
  4. Check network connectivity

3. Duplicate Records

Problem: Same record appears multiple times
Solution:
  1. Review matching criteria settings
  2. Check for duplicate records in source system

Error Codes

Error Code Description Solution
XE001 Authentication failure Reconnect to Xero
XE002 Invalid account mapping Check account mappings
XE003 Missing required field Populate required fields
XE004 Rate limit exceeded Wait and retry
XE005 Network timeout Check internet connection

Frequently Asked Questions

General Questions

How often does the sync occur?

Sync frequency depends on your configuration. Options include real-time, hourly, daily, or manual sync.

Can I sync historical data?

Yes, use the bulk sync feature to synchronize historical records within a specified date range.

What happens if I modify a record in both systems?

The system uses your configured conflict resolution rules. By default, the most recently modified record takes precedence.

Is there a limit to how much data can be synced?

Xero API has rate limits. The integration handles these automatically by queuing requests.

Technical Questions

Can I customize field mappings?

Yes, go to Xero Settings → Field Mappings to customize how fields are mapped between systems.

How do I handle multi-currency transactions?

The integration supports multi-currency if enabled in both systems. Exchange rates are synchronized from Xero.

Can I exclude certain records from syncing?

Yes, use sync filters to exclude records based on specific criteria.

Troubleshooting Questions

What if sync fails for a specific record?

Check the Integration Log for error details. Most issues are related to missing required fields or validation errors.

How do I reset the integration?

Go to Xero Settings → Advanced → Reset Integration. This will clear all mappings and require reconfiguration.

Can I pause synchronization temporarily?

Yes, disable the integration in Xero Settings. Re-enable when ready to resume syncing.

Best Practices

Data Management

  1. Regular Backups: Always backup data before major sync operations
  2. Test Environment: Test configurations in a staging environment first
  3. Monitor Logs: Regularly review integration logs for issues
  4. Clean Data: Ensure data quality in both systems before syncing

Performance Optimization

  1. Sync Scheduling: Schedule heavy sync operations during off-peak hours
  2. Selective Sync: Only sync necessary data to improve performance
  3. Regular Maintenance: Periodically clean up old log entries
  4. Monitor Resources: Keep an eye on system resources during sync operations

Security

  1. Access Control: Limit integration access to authorized users only
  2. Regular Updates: Keep the integration app updated
  3. Audit Trail: Regularly review sync logs for unusual activity
  4. Secure Credentials: Ensure Xero credentials are properly secured
